# Schedule forge
Small app for creating personal tasks with small rewards. Overlays on top of all over apps.

## How to use:
Download the file and launch "Schedule forge.exe"

## Main features:
  -Click on any of the 5 "Forges" to assign a new task.\
  -Assign the time, name and reminder period to the task.\
  -The task will change color when its time will come.\
  -Clicking on the task will mark them as completed and award you with a small amount of currency.\
  -Currency can be spent in the app's shop to buy small cosmetic features.\
